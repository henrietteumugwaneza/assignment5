# AUCA Campus Information Website

# Project Description
This project is a responsive single-page campus information website designed for the Adventist University of Central Africa (AUCA).  
The website presents important campus services, locations, announcements, and layout demonstrations for students.

The main goal of this project is to demonstrate proper use of HTML structure, CSS layout techniques, positioning, and responsive design.

---

# Features and Layout Overview
The website includes the following key sections and behaviors:

# Navigation Bar
- Displays the campus name (AUCA)
- Contains multiple navigation links
- Includes a dropdown menu
- Uses internal links to navigate to different sections of the page
- Remains usable on all screen sizes

# Introduction Section
- Page title and short description
- Unordered list explaining the type of campus information available
- Responsive text that adapts to different screen sizes

# Campus Services Section
- Service cards (Library, IT Support, Health Center, Chaplaincy)
- Cards arranged using CSS Grid
- Layout adjusts from multiple columns to single column on small screens

# Campus Locations Section
- Gallery-style layout displaying campus facilities
- Uses images arranged in a responsive grid
- Images resize properly without distortion

# Announcements & Notices
- Important campus announcements displayed clearly
- Includes a sticky notice that remains visible while scrolling
- Notices are visually highlighted

# Display Behavior Section
- Demonstrates different display behaviors
- Includes elements that change visibility on smaller screens
- Uses CSS media queries to hide content when necessary

---

# Responsiveness
The website is fully responsive and adapts to:
- Desktop screens
- Tablet screens
- Mobile screens

Responsive techniques used include:
- CSS Grid
- Flexbox
- Media queries
- Relative units

---

# Project Files
- `index.html` – Main HTML file containing all page sections
- `style.css` – External CSS file for styling and responsiveness
- `README.md` – Project documentation
